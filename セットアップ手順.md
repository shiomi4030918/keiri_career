# 経理キャリア LP セットアップ手順

## ファイル構成
- `index.html` … LP本体（リスティング広告の遷移先）
- `Code.gs` … Google Apps Script（応募データをスプレッドシート保存＋shiomiaki0918403@gmail.comへメール通知）
- `images/hero-bg.png` … ヒーロー背景画像（製造JOBから流用。差し替え可）
- `favicon.png` … ファビコン

## GAS（スプレッドシート保存＋メール送信）の接続手順
1. Google ドライブで保存用スプレッドシートを1枚作成し、URLの
   `https://docs.google.com/spreadsheets/d/【ここがID】/edit` の ID をコピー。
2. [script.google.com](https://script.google.com) で新規プロジェクトを作成し、`Code.gs` の中身を貼り付け。
3. `Code.gs` 冒頭の `SPREADSHEET_ID` にコピーしたIDを貼り付け（空のままだと初回実行時に自動作成され、IDがログに出ます）。
4. エディタ上部の関数選択で `testRun` を選び ▶ 実行 → 初回は権限の承認。スプレッドシートに行が追加され、shiomiaki0918403@gmail.com にメールが届けば成功。
5. 「デプロイ > 新しいデプロイ > 種類：ウェブアプリ」
   - 次のユーザーとして実行：**自分**
   - アクセスできるユーザー：**全員**
   発行された「ウェブアプリのURL」をコピー。
6. `index.html` の `var GAS_URL = "";` の `""` の中にそのURLを貼り付けて保存。

> メール送信先は `Code.gs` の `NOTIFY_TO`（現在 `shiomiaki0918403@gmail.com`）で変更できます。
> 設問を追加・変更した場合は、`index.html` の `labels`/`order` と `Code.gs` の `FIELDS` を合わせてください。

## フォーム構成（全7ステップ）
1. 現在の就業状況
2. 経理の経験年数
3. 保有資格
4. 希望の勤務形態 ＋ 上場企業での経理経験
5. 勤務エリア（地方 → 都道府県 → 市区町村／各都道府県の先頭は「全域」）
6. お名前・生まれ年
7. 電話番号・メールアドレス・プライバシーポリシー同意 → 送信

※ トップの「近いうちに転職したい」「どんな求人があるか見たい」はどちらも同じフォームへ進みます（押した内容は「きっかけ」として記録）。
